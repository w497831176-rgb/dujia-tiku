# 版本历史

## 版本追溯说明

公开仓库从 V1.2.0 的完整代码快照开始建立。V1.0.0、V1.1.0 已经真实经历过产品迭代，但没有可迁移的独立 Git 提交，因此以下内容根据原始 PRD、复刻手册和当前代码回溯整理，不伪造旧提交或旧标签。

## V1.0.0 — 基础刷题系统

目标是形成可登录、可答题、可记录学习进度的最小可用版本。

- 注册与登录
- 题库及单题展示
- 选项作答与正确答案
- 答题记录、进度和正确率
- 错题集
- 移动端响应式页面

## V1.1.0 — 学习增强与 AI

目标是从“能刷题”扩展到“能理解、能复习”。

- 题目解析、错误辨析和知识点回顾
- 举一反三变式题
- 查看全部与只看未做模式
- 答题后停留当前题
- DeepSeek AI 提炼
- 提炼全集与提炼错题
- 邀请记录和权益状态
- 音效及移动端交互限制

## V1.2.0 — 环境隔离与工程化

目标是让测试、部署和发布过程可重复、可验证。

- 测试与生产环境完全隔离
- 独立 Docker 容器、网络、端口、环境变量和数据库
- API、健康检查和浏览器自动化测试
- 部署与测试环境初始化脚本
- 统一数据库文件名
- 建立更新日志和版本发布规范
- 修复只看未做与举一反三相关状态问题

V1.2.0 是当前公开仓库的首个完整基线，应在 Git 中标记为 `v1.2.0`。
