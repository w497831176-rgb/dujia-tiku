# UI 像素与交互基线

## 总体布局

- 页面基准宽度为 480px，桌面端居中并显示轻阴影。
- 页面背景为 `#f0f4f8`，应用容器背景为 `#f8fafc`。
- 字体栈依次使用系统字体、Segoe UI、苹方、冬青黑体和微软雅黑。
- 主色为蓝色系：`#3b82f6` / `#2563eb`；强调色为橙色 `#f97316`。
- 卡片左右外边距 16px、圆角 12px、内边距 16px。

## 登录与注册

- 未登录时只显示 `#loginPage`。
- 标题区、输入框、主按钮与登录/注册链接保持居中单列。
- 输入框圆角 12px，聚焦边框为蓝色，并带 3px 半透明焦点环。
- 登录成功后隐藏登录页，显示 `#mainApp` 并默认进入题库。

## 题库页

- 顶部依次为标题/重置入口、进度条、查看全部/只看未做切换。
- 题型标签颜色：单选蓝、多选紫、判断绿、填空橙。
- 选项按钮边框 1.5px、圆角 10px、垂直间距 10px。
- 答对时绿色闪烁并显示勾；答错时所选项红色闪烁，同时高亮正确项。
- 答题后依次出现正确答案、解题思路、错误辨析、知识点回顾和举一反三。
- “只看未做”模式答题后仍停留当前题，下一题按钮由禁用变为可用。

## 错题、提炼与我的

- 错题列表显示题型、题干摘要和累计错误次数，可进入详情并移除单题。
- 提炼页分“提炼全集”和“提炼错题”两个子标签；报告详情使用 Markdown 表格。
- 我的页显示用户名、邀请码、答题统计、购买状态、邀请数据及账户操作。
- 原版付费弹层及测试版自动核销文案和行为暂时保留。

## 固定操作区

- 底部四个 Tab 固定在视口底部，最大宽度 480px。
- 题库上一题/下一题操作条固定在 Tab 上方。
- 当前 Tab 使用 `#2563eb`，非当前 Tab 使用 `#94a3b8`。

## 不可破坏的交互

- 回答前下一题按钮在“只看未做”模式禁用。
- 回答后解析留在当前题，不自动跳转。
- 变式题的答案和解析必须属于同一道变式题。
- 重置答题记录不清空错题集。
- 页面仍保留原版右键、复制、快捷键和移动端缩放限制。
- 后续改动须先在 375×812 与 480px 宽度下验证，再发布测试环境。
