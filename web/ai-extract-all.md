### 3. SPSS — 统计分析软件

| 知识点                   | 答案/速记口诀                                                                   |
| -------------------------- | --------------------------------------------------------------------------------- |
| **两大视图**       | 📌​**数据视图**​（看数据本身）/​**变量视图**​（看变量属性定义） |
| 💣 易错                  | 变量视图定义属性，数据视图查看实际数据，别搞反                                  |
| **变量类型**       | 数值型(Numeric) / 字符串型(String) / 日期型(Date)                               |
| **集中趋势**       | 📌 均值(Mean) / 中位数(Median) / 众数(Mode)                                     |
| **离散趋势**       | 📌 方差(Variance) / 标准差(Std.Deviation) / 极差(Range)                         |
| 方差                     | 数据偏离均值程度的平均值                                                        |
| 标准差                   | 方差的平方根，与数据同单位                                                      |
| 极差                     | 最大值 - 最小值                                                                 |
| **偏度(Skewness)** | &gt;0 右偏（右尾长）/ &lt;0 左偏（左尾长）/ =0 对称                                   |
| **峰度(Kurtosis)** | &gt;0 尖峰（集中）/ &lt;0 扁平（分散）/ =0 正态                                       |
| **菜单路径**       |                                                                                 |
| 描述统计                 | 分析 → 描述统计 → 频率/描述                                                   |
| 相关分析                 | 分析 → 相关 → 双变量                                                          |
| 回归分析                 | 分析 → 回归 → 线性                                                            |
| 因子分析                 | 分析 → 降维 → 因子                                                            |
| 聚类分析                 | 分析 → 分类 → K-均值聚类/系统聚类                                             |
| 💣 易错                  | 因子分析在"降维"菜单下，不在"分类"下                                            |

### 4. Docker — 容器化部署

| 知识点                               | 答案/速记口诀                                                                                                   |
| -------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| **三大概念**                   | 📌​**镜像**​(Image=模板) /​**容器**​(Container=运行实例) /​**仓库**​(Registry=存储镜像) |
| 类比                                 | 镜像=类，容器=对象实例                                                                                          |
| **Dockerfile 指令**            |                                                                                                                 |
| `FROM`                           | 指定基础镜像（​**必须第一条**​）                                                                        |
| `RUN`                            | 在构建时执行命令（安装软件等）                                                                                  |
| `CMD`                            | 容器启动时执行的​**默认命令**​（可被覆盖）                                                              |
| `COPY`                           | 复制本地文件到镜像                                                                                              |
| `ADD`                            | 类似 COPY，但支持 URL 和自动解压 tar                                                                            |
| `EXPOSE`                         | 声明容器监听端口（仅声明，不映射）                                                                              |
| 💣 易错                              | COPY 不支持 URL，ADD 支持；CMD 可被覆盖，ENTRYPOINT 不可                                                        |
| **常用命令**                   |                                                                                                                 |
| `docker pull 镜像`               | 从仓库拉取镜像                                                                                                  |
| `docker run 镜像`                | 创建并运行容器                                                                                                  |
| `docker ps`                      | 查看**运行中**的容器                                                                                      |
| `docker ps -a`                   | 查看**所有**容器（含已停止）                                                                              |
| `docker images`                  | 查看本地镜像列表                                                                                                |
| `docker stop 容器ID`            | 停止容器                                                                                                        |
| `docker rm 容器ID`              | 删除容器                                                                                                        |
| `docker rmi 镜像ID`             | 删除镜像（💣 注意 rm 删容器，rmi 删镜像）                                                                       |
| `docker build -t 名称 .`         | 根据 Dockerfile 构建镜像                                                                                        |
| `docker exec -it 容器 /bin/bash` | 进入运行中容器的终端                                                                                            |
| `docker logs 容器ID`            | 查看容器日志                                                                                                    |

### 6. Excel 进阶功能

#### 📌 常用函数

| 函数                                 | 功能                         | 速记                              |
| -------------------------------------- | ------------------------------ | ----------------------------------- |
| `SUM(范围)`                      | 求和                         | 加总                              |
| `AVERAGE(范围)`                  | 求平均值                     | 算术平均                          |
| `MAX(范围)`                      | 最大值                       |                                   |
| `MIN(范围)`                      | 最小值                       |                                   |
| `COUNT(范围)`                    | 计数（​**仅数字**​） | 💣 只数数字单元格                 |
| `COUNTA(范围)`                   | 计数（​**非空**​）   | 💣 数所有非空单元格               |
| `IF(条件, 真值, 假值)`           | 条件判断                     | 三路分支                          |
| `VLOOKUP(查找值, 范围, 列号, 0)` | **垂直**查找           | 💣 按列查找，第 4 参数 0=精确匹配 |
| `HLOOKUP(查找值, 范围, 行号, 0)` | **水平**查找           | 按行查找                          |
| `LEFT(文本, n)`                  | 取左边 n 个字符              |                                   |
| `RIGHT(文本, n)`                 | 取右边 n 个字符              |                                   |
| `MID(文本, 起始, n)`             | 取中间 n 个字符              |                                   |
| `DATE(年,月,日)`                 | 生成日期                     |                                   |

> 💣 ​**VLOOKUP**​：第 4 参数填 **0 或 FALSE** 表示精确匹配，填 1 或 TRUE 表示近似匹配
> 💣 ​**COUNT vs COUNTA**​：COUNT 只计数字，COUNTA 计所有非空

#### 📌 单元格引用

| 引用类型           | 写法                                        | 拖动时变化规则 |
| -------------------- | --------------------------------------------- | ---------------- |
| **相对引用** | `A1`                                    | 行列都变       |
| **绝对引用** | `$A$1`                                  | 行列都不变     |
| **混合引用** | `$A1`（列锁行变）/`A$1`（行锁列变） | 锁谁谁不变     |

> 📌 速记口诀：**\$ 锁谁，谁不动**

### 7. TensorFlow / Keras 深度学习

| 知识点                          | 答案/速记口诀                                                               |
| --------------------------------- | ----------------------------------------------------------------------------- |
| **创建模型**              | `model = tf.keras.Sequential()`                                         |
| **添加卷积层(图像)**      | `tf.keras.layers.Conv2D(filters, kernel_size, activation, input_shape)` |
| **添加卷积层(序列)**      | `tf.keras.layers.Conv1D(...)`— 处理语音/文本                           |
| **添加全连接层**          | `tf.keras.layers.Dense(单元数, activation)`                             |
| **添加 Dropout 层**       | `tf.keras.layers.Dropout(丢弃概率)`                                     |
| **展平层**                | `tf.keras.layers.Flatten()`— 多维 → 一维                              |
| **model.fit()**           | 📌 功能=​**模型训练**​（不是评估！）                                |
| **model.evaluate()**      | 📌 功能=​**模型评估**​，返回损失 + 指标                             |
| **model.predict()**       | 功能=推理预测，只返回预测值                                                 |
| **model.compile()**       | 配置优化器、损失函数、指标                                                  |
| **model.summary()**       | 查看模型结构                                                                |
| **validation\_split=0.2** | 📌 从**训练数据**中分出\*\*20%\*\*作验证集（💣 不是全部数据）         |
| **evaluate 参数**         | `model.evaluate(x_test, y_test)`— 传**测试集**的特征和标签       |
| **输入神经元计算**        | input\_shape 各维度​**相乘**​：如(28,28,1)→784                     |
| **Dropout(0.25)**         | 丢弃概率 25%，**保留概率 75%**                                        |

> 💣 ​**Dropout 不删除数据**​，只随机关闭神经元
> 💣 ​**fit 是训练，evaluate 是评估，predict 是预测**​——三者不要混淆
> 💣 ​**增加迭代次数不能优化模型结构**​，可能过拟合

#### 📌 激活函数

| 函数                     | 公式/特点                                  | 适用场景                  |
| -------------------------- | -------------------------------------------- | --------------------------- |
| **Sigmoid**        | 1/(1+e⁻ ˣ)，输出(0,1)                    | 二分类输出层              |
| **Tanh**           | 输出(-1,1)                                 | 隐藏层                    |
| **ReLU**           | max(0,x)，📌 最常用                        | 隐藏层（💣 可能死神经元） |
| **Softmax**        | 输出概率分布，和=1                         | 📌 **多分类**输出层  |
| 💣 **不是**激活函数 | ​**Cos 函数**​、**sin 函数** | 周期函数不适合            |

### 5. 机器学习基础

#### 📌 三种学习范式

| 类型                   | 特点                             | 典型算法                                                             |
| ------------------------ | ---------------------------------- | ---------------------------------------------------------------------- |
| **监督学习​**   | 有标签数据，学习输入 → 输出映射 | 线性回归、逻辑回归、决策树、SVM、朴素贝叶斯、随机森林、KNN、神经网络 |
| **无监督学习​** | 无标签数据，发现数据内在结构     | 📌K-Means 聚类、层次聚类、PCA 降维                                   |
| **强化学习​**   | 通过与环境交互获得奖励来学习策略 | Q-Learning、Policy Gradient                                          |

#### 📌 过拟合 vs 欠拟合

|                  | 过拟合                                  | 欠拟合                             |
| ------------------ | ----------------------------------------- | ------------------------------------ |
| **定义**   | 训练误差低，测试误差高                  | 训练误差高，测试误差也高           |
| **原因​** | 模型太复杂，记住噪声                    | 模型太简单，学不到规律             |
| **解决**   | 📌Dropout/正则化/早停/增加数据/简化模型 | 增加模型复杂度/增加特征/减少正则化 |

#### 📌 评估指标公式

| 指标                        | 公式              | 含义                                             |
| ----------------------------- | ------------------- | -------------------------------------------------- |
| **准确率(Accuracy)**  | (TP+TN) / 全部    | 整体预测正确比例                                 |
| **精确率(Precision)** | TP / (TP+FP)      | 预测为正的里面，真正为正的比例                   |
| **召回率(Recall)**    | 📌TP / (TP+FN)    | 真正为正的里面，被找出来的比例                   |
| **F1-score**          | 📌2×P×R / (P+R) | 精确率和召回率的​**调和平均**​，范围 0-1 |
| **AUC**               | ROC 曲线下面积    | 越大越好，不受阈值影响                           |

> 💣 **减少误识别（FP 多）→ 关注精确率**
> 💣 **减少漏识别（FN 多）→ 关注召回率**
> 💣 ​**F1 公式**​：2×0.8×0.4/(0.8+0.4) ≈ 0.53（要会算！）
> 💣 ​**混淆矩阵**​：FP=假正类（误识别），FN=假负类（漏识别）

### 6. 归一化 vs 标准化

|                | 归一化(Min-Max)                     | 标准化(Z-score)            |
| ---------------- | ------------------------------------- | ---------------------------- |
| **公式** | (X-Xmin)/(Xmax-Xmin)                | (X-均值)/标准差            |
| **结果** | 📌 映射到\*\*[0,1]\*\*              | 📌 均值 0，标准差 1        |
| **适用** | 无异常值，需要固定范围              | 有异常值，保持分布         |
| **代码** | `(Z-Z.min())/(Z.max()-Z.min())` | `(Z-Z.mean())/Z.std()` |

> 💣 聚类前需**归一化**消除量纲差异
> 💣 未归一化 → **梯度更新不稳定，收敛缓慢**
