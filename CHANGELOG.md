# 更新日志

本项目采用语义化版本号。V1.0.0 与 V1.1.0 的内容根据原始 PRD 和现有产品行为回溯整理；公开 Git 仓库从 V1.2.0 完整快照开始建立可追溯提交。

## [1.2.0] - 2026-07-03

### 工程化

- 建立相互隔离的测试与生产 Docker 环境
- 分离容器、网络、数据卷、环境变量和 SQLite 数据库
- 增加 API、健康检查和浏览器端到端测试脚本
- 增加生产部署及测试环境初始化脚本
- 统一数据库文件名为 `du-tiku.db`
- 建立 CHANGELOG、版本号和发布规范

### 交互优化

- 新增「查看全部 / 只看未做」刷题模式
- 只看未做模式中，答题后停留当前题并展示完整解析
- 当前题未回答时禁用下一题按钮，回答后恢复
- 进度条显示已做或未做题目进度
- 题号统一使用全部题库中的固定序号
- 增加答题音效、防复制限制和重置二次确认

### 修复

- 修复只看未做模式答题后自动跳题
- 修复进入下一题后按钮状态未重置
- 修复举一反三解析与题干错配

## [1.1.0] — 回溯整理

### 学习能力增强

- 新增题目解析、错误选项辨析和知识点回顾
- 新增「举一反三」变式题及即时解析
- 接入 DeepSeek API 生成 AI 提炼报告
- 增加错题提炼和题库知识点全集
- 增加邀请记录、权益状态和演示版自动核销流程
- 增加答题正确/错误音效及移动端交互限制

- 题库列表及单题刷题

- 答题进度、正确率和重置记录
